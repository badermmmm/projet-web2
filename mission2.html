Requete 1

  • Enoncé/Interpretation:
    • Pour optimiser les livraisons, on regarde quels couples de clients sont à Paris. Lister les possibilités avec pour chacun Code, société et mail. Il faudra donc extraire tout les clients qui habitent à Paris et faire des couples de clients avec leurs code client, nom de société et l'adresse mail
  • Résultat:
  • Code Sql:

    • SELECT C1.CodeCli AS CodeClient1, C1.Societe AS Societe1, C1.MailCli AS Mail1, C2.CodeCli AS CodeClient2, C2.Societe AS Societe2, C2.MailCli AS Mail2

    FROM Clientele AS C1, Clientele AS C2

    WHERE C1.Ville = "Paris" AND C2.Ville = "Paris" AND C1.CodeCli < C2.CodeCli

    ORDER BY C1.CodeCli, C1.Societe, C1.MailCli, C2.CodeCli, C2.Societe, C2.MailCli;

    SELECT C1.CodeCli AS CodeClient1, C1.Societe AS Societe1, C1.MailCli AS Mail1, C2.CodeCli AS CodeClient2, C2.Societe AS Societe2, C2.MailCli AS Mail2

    FROM Clientele AS C1, Clientele AS C2

    WHERE C1.Ville = "Paris" AND C2.Ville = "Paris" AND C1.CodeCli < C2.CodeCli

    ORDER BY C1.CodeCli, C1.Societe, C1.MailCli, C2.CodeCli, C2.Societe, C2.MailCli;
